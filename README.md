# Mini Linux Shell

##  Español

Mini terminal de Linux desarrollada en **C++** que permite ejecutar comandos del sistema desde una interfaz simple en consola.

### Características

- Ejecución de comandos del sistema
- Comando interno `cd`
- Gestión de procesos usando `fork()` y `execvp()`
- Interfaz de línea de comandos

### Tecnologías utilizadas

- C++
- Programación de sistemas en Linux
- Llamadas al sistema (`fork`, `exec`, `wait`)

### Compilación

make

### Ejecución

./shell

##  English

A simple Linux shell developed in **C++** that allows executing system commands through a basic command-line interface.

### Features

- System command execution
- Built-in `cd` command
- Process management using `fork()` and `execvp()`
- Command-line interface

### Technologies

- C++
- Linux system programming
- System calls (`fork`, `exec`, `wait`)

### Build

make

### Run

./shell
